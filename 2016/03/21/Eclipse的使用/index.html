Eclipse的使用

罗炜光 3月 21, 2016

常用

  • Ctrl + Shift + F 格式排版
  • Ctrl + 1 快速修正
  • Ctrl + / 注释
  • Ctrl + Shift + / 添加块注释
  • Ctrl + Shift + \ 除去块注释
  • Ctrl + Shift + Space 方法参数提示
  • Ctril + F6 页面切换
  • Ctrl + Shift + O 去掉没用到的包
  • F2 显示提示
  • F4 打开元素继承列表
  • Ctrl + 鼠标点击 查看源码
  • Ctrl + Z 撤销
  • Ctrl + Y 重做

跳转查询

  • Ctrl + T 列出接口的实现类列表或继承的父类
  • F3 查看源码
  • Ctrl + O 查看当前文件的结构
  • Ctrl + Q 回到最后一次编辑的地方
  • Ctrl + Alt + H 打开调用层次结构
  • Ctrl + Shift + R 打开资源
  • Ctrl + Shift + T 打开类型

搜索

  • Ctrl + H 打开搜索对话框
  • Ctrl + F 打开文本搜索对话框
  • Ctrl + G 在workspace中搜索选中元素的声明
  • Ctrl + Alt + G 在workspace中搜索选中的文本
  • Ctrl + Shift + G 在workspace中搜索选中元素的引用

编辑页面操作

  • Ctrl + W 关闭当前页面
  • Ctrl + E 查看所有页面目录
  • Alt + ← 前一个编辑的页面
  • Alt + → 后一个编辑的页面
  • Ctrl + ↑ 上滚行
  • Ctrl + ↓ 下滚行
  • Ctrl + Shift + - 双编辑窗口

文本操作

  • Ctrl + D 删除行:
  • Tab 把当前选中的文本向右推进
  • Shift + Tab 把当前选中的文本向左缩进
  • Ctrl + Shift + Y 把当前选中的文本全部变为小写
  • Ctrl + Shift + X 把当前选中的文本全部变为大写
  • Alt + ↑ 将行上移
  • Alt + ↓ 将行下移
  • Ctrl + Shift + Enter 在当前行上插入一行
  • Shift + Enter 在当前行下插入一行

重构

  • Alt + Shift + T 打开重构菜单
  • Alt + Shift + R 重命名
  • Alt + Shift + M 选中内容生成一个方法
  • Alt + Shift + V 移动
  • Alt + Shift + C 改变方法参数
  • Alt + Shift + L 抽取本地变量

调试

  • F5 单步跳入
  • F6 单步跳过
  • F7 单步返回
  • F8 继续
  • Ctrl + Shift + D 显示变量的值
  • Ctrl + Shift + B 在当前行设置或者去掉断点
  • Ctrl + R 运行至行

设置字体与字体大小

Window -> Preferences -> General -> Appearance -> Colors and Fonts右边Basic中的TextFont，再点击Edit进行修改

java智能提示

  1. Window -> Preferences
  2. Java －> Editor －> Content Assist
  3. 右侧的”Auto-Activation”找到”Auto Activation triggers for java”选项
  4. 填入会提示的字符，如”.abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZ”

XML智能提示

  1. Window -> Preferences
  2. XML －> XML Files -> Editor －> Content Assist
  3. 右侧的”Auto-Activation”找到”Prompt when these characters are inserted “选项
  4. 填入会提示的字符，如”<=:.abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZ”

设置默认编码

  1. Window -> Preferences -> General -> Workspace 面板Text file encoding

设置代码风格

  1. Window -> Preferences
  2. Java -> Code Style -> Formatter
  3. 点击面板的Edit
  4. 在Profile name:中输入一个名称
  5. 在下面的设置中设置属性，如果是设置换行，点击Braces设置，完成后点击OK

安装主题（Color Theme）

  1. Help->Install New Software
  2. 在Work with中输入http://eclipse-color-theme.github.io/update/
  3. 下方会出现Eclipse Color Theme,在项目前点勾
  4. 点击Next >,再点Next >
  5. 选择I accept the terms of the license agreements
  6. 点击Finish

导入JavaDOC与源码

  1. Window -> Preferences
  2. Java -> Installed JREs
  3. 选中使用的Jre包，点击Edit
  4. Ctrl + A 全选中Jre包
  5. 点击Source Attachment..
  6. 点击External location,点击External File
  7. 选择Jdk文件夹下的src.zip文件
  8. 点击打开，点击OK
